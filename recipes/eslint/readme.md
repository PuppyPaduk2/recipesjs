# ESLint

## TS

* Install dependencies

```sh
npm install --save-dev \
  @typescript-eslint/eslint-plugin@^5.13.0 \
  @typescript-eslint/parser@^5.13.0 \
  eslint@^8.10.0 \
  eslint-config-prettier@^8.5.0 \
  eslint-plugin-prettier@^4.0.0 \
  prettier@^2.5.1 \
  typescript@^4.6.2
```

* Copy eslint/ts/.eslintrc.js
